# Bachelor Project

Project consists of the Docker Compose file in this repository and the 2 services in the following repositories:

[Inventory Service](https://github.com/tomsten2000/InventoryService) &nbsp; [Trade Service](https://github.com/tomsten2000/TradeService)

### About
This project aims to create a third party marketplace for Steam items surcumventing the high fees of using Steam Community Market (Steam's first party trading platform) by using the trade functions within Steam and a virtual currency used for buying and selling on the platform.

### Setup:
```console
git clone https://github.com/tomsten2000/BachelorProjekt.git;
cd BachelorProjekt;
git clone https://github.com/tomsten2000/InventoryService.git;
git clone https://github.com/tomsten2000/TradeService.git;
docker-compose up;
```
